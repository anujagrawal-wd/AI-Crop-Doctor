
# 🌱 AI Crop Doctor

AI Crop Doctor is an AI-powered crop disease diagnosis platform that helps farmers identify plant diseases using a smartphone camera.

## Vision

To provide every farmer with an intelligent assistant capable of:

- Detecting crop diseases
- Assessing disease severity
- Suggesting treatment
- Recommending preventive measures
- Supporting multiple Indian languages
- Working offline

## Tech Stack

- Python
- TensorFlow
- OpenCV
- FastAPI
- Flutter
- PostgreSQL
